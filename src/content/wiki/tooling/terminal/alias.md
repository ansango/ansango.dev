# Aliases

Todos tienen sus alias favoritos para git y la terminal. He escrito sobre mis alias de git anteriormente, pero no sobre los alias de mi terminal.

Actualmente, mi terminal preferida es zsh. Esto es lo que tengo actualmente en mi archivo de configuración `.zshrc`

```shell
alias rimraf='rm -rf'
alias nr='npm run'
alias ni='npm i'
alias '$'=''
alias zshconfig='nano ~/.zshrc'
```
