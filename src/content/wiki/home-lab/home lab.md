# Home Lab

Esta sección contiene toda la documentación relacionada con mi Home Lab, incluyendo la configuración de servicios auto-alojados (self-hosted) y otras experimentaciones.

## Self-Hosted

Aquí se detallan los servicios que mantengo en mi infraestructura:

- **Gestión de Contenedores**:
    - [[portainer|Portainer]]: Gestión de entornos Docker.
    - [[dockge|Dockge]]: Interfaz de usuario para Docker Compose.

- **Proxy Inverso y Redes**:
    - [[nginx-proxy-manager|Nginx Proxy Manager]]: Gestión de proxy inverso con interfaz web.
    - [[traefik|Traefik]]: Proxy inverso nativo de la nube.
    - [[pihole|Pi-hole]]: Bloqueador de anuncios a nivel de red.
    - [[adguard-home|AdGuard Home]]: Bloqueador de anuncios a nivel de red.
    - [[flaresolverr|FlareSolverr]]: Servidor proxy para eludir protecciones de Cloudflare.

- **VPN y Acceso Remoto**:
    - [[tailscale|Tailscale]]: Red VPN de malla basada en WireGuard.
    - [[wireguard|WireGuard]]: VPN rápida y moderna.

- **Almacenamiento y Colaboración**:
    - [[nextcloud|Nextcloud]]: Nube privada para archivos y colaboración.
    - [[filebrowser|FileBrowser]]: Gestor de archivos web.

- **Multimedia**:
    - [[jellyfin|Jellyfin]]: Servidor de medios de código abierto.
    - [[gonic|Gonic]]: Servidor de streaming de música.
    - [[qbittorrent|qBittorrent]]: Cliente BitTorrent.
    - [[booklore|BookLore]]: Gestor de libros y cómics.

- **Seguridad y Administración**:
    - [[vaultwarden|Vaultwarden]]: Gestor de contraseñas auto-alojado.
    - [[duplicati|Duplicati]]: Copias de seguridad cifradas.
    - [[dozzle|Dozzle]]: Visor de logs de Docker en tiempo real.

- **Downloaders**:
    - [[jdownloader-2|JDownloader 2]]: Gestor de descargas.
    - [[indexers|Indexers]]: Stack de indexadores y gestión de medios.

- **Bookmarks**:
    - [[hoarder|Hoarder / Karakeep]]: Gestor de marcadores con IA.
    - [[linkwarden|Linkwarden]]: Gestor de marcadores colaborativo.

- **Wikis**:
    - [[docmost|Docmost]]: Wiki y documentación colaborativa.

- **Dashboards**:
    - [[glance|Glance]]: Dashboard auto-alojado.

- **DevOps**:
    - [[gitea|Gitea]]: Servicio de Git auto-alojado.

- **Monitoring**:
    - [[monitoring|Stack de Monitoreo]]: Prometheus y Grafana con Node Exporter.
