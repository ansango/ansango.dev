# Beets

Esta herramienta nos permite organizar muchos metadatos de forma automatica con la terminal, tenemos que tener instalado py, lo recomendable es instalar py como entorno para esta herramienta, de otra manera si lo instalamos con arch mediante pacman instalar algunos plugins sera imposible y tendremos si o si que usar un entorno en py, asiq es mejor organizarlo directamente en py
