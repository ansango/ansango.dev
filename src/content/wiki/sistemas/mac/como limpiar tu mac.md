# Cómo limpiar tu mac

## OnyX

Onyx es una herramienta de utilidad para macOS que ofrece varias opciones para personalizar y optimizar el rendimiento del sistema operativo. Las funciones más destacadas de Onyx incluyen:

- **Opciones de sistema**: Ajustar parámetros como memoria, papel y pantalla, y sistema de archivos.
- **Opciones de seguridad**: Activar o desactivar el firewall y configurar las actualizaciones de software.
- **Opciones de privacidad**: Eliminar el historial de navegación y registros de sistema.
- **Opciones de rendimiento**: Ajustar la optimización de memoria, limpiar la caché y mejorar el rendimiento del sistema.

Al utilizar Onyx, puedes personalizar y optimizar el rendimiento de tu sistema operativo para mejorar su estabilidad y velocidad.

## Pearcleaner

Pearcleaner es una aplicación gratuita y de código abierto para Mac que funciona como un limpiador de aplicaciones. Sus principales características incluyen:

1. Búsqueda de archivos huérfanos de aplicaciones previamente desinstaladas.
2. Limpieza de archivos y caché de entornos de desarrollo.
3. Monitor Sentinel que vigila la papelera para limpiar archivos de aplicaciones eliminadas.
4. Modo mini activable desde la configuración.
5. Opción de icono en la barra de menú.
6. Soporte para línea de comandos (CLI).
7. Compatibilidad con arrastrar y soltar aplicaciones.
8. Extensión opcional para Finder que permite desinstalar aplicaciones directamente.
9. Sistema de temas con selector de colores personalizado.
10. Diferenciación entre aplicaciones regulares, web-apps de Safari y aplicaciones móviles.
11. Opción de desinstalación limpia para la propia aplicación Pearcleaner.
12. Exportación de paquetes de aplicaciones para migrar apps y su caché a un nuevo sistema.
13. Limpieza opcional de Homebrew.

Pearcleaner está diseñado para ofrecer una alternativa de código abierto a otros limpiadores de aplicaciones, con un enfoque en la privacidad y la personalización.
