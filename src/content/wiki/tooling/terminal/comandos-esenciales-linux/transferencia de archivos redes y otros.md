# Transferencia de Archivos y Redes y otros comandos

## Transferencia de Archivos y Redes

- `scp`: Copia archivos de forma segura entre hosts.
- `rsync`: Utilidad de sincronización de archivos remotos.
- `wget`: Recupera archivos de la web usando HTTP, HTTPS y FTP.
- `curl`: Transfiere datos con URLs.
- `ssh-keygen`: Genera claves SSH para autenticación segura.
- `ssh-copy-id`: Copia las claves SSH al archivo authorized_keys de un servidor remoto.

## Misceláneos

- `history`: Muestra el historial de comandos.
- `alias`: Crea alias de comandos.
- `watch`: Ejecuta un programa periódicamente, mostrando la salida a pantalla completa.
- `man`: Muestra las páginas del manual de los comandos.
