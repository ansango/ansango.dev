# Commands adicionales en yay

Para imprimir estadísticas de paquetes y estado del sistema con Yay, ejecute el siguiente comando:

```shell
yay -Ps
```

En caso de que necesite ayuda con los comandos, puede obtener fácilmente ayuda en la línea de comandos usando el indicador --help:

```shell
yay --help
```

Alternativamente, puede ver la página del manual de Yay usando:

```shell
man yay
```
