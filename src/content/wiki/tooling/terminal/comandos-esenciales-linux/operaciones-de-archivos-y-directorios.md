# Operaciones de Archivos y Directorios

- `ls`: Lista el contenido de un directorio.
- `cd`: Cambia de directorio.
- `pwd`: Muestra el directorio de trabajo actual.
- `mkdir`: Crea un directorio.
- `rmdir`: Elimina un directorio.
- `touch`: Crea un archivo vacío o actualiza la marca de tiempo de un archivo existente.
- `cp`: Copia archivos o directorios.
- `mv`: Mueve o renombra archivos o directorios.
- `rm`: Elimina archivos o directorios.
- `cat`: Concatena y muestra el contenido de un archivo.
- `less`: Muestra el contenido de un archivo con paginación.
- `head`: Muestra el principio de un archivo.
- `tail`: Muestra el final de un archivo.
- `ln`: Crea enlaces entre archivos.
- `find`: Busca archivos y directorios.
- `grep`: Busca patrones en archivos.
- `locate`: Encuentra archivos por nombre.
- `diff`: Compara archivos línea por línea.
- `wc`: Cuenta palabras, líneas y caracteres en un archivo.
- `chmod`: Cambia los permisos de un archivo.
- `chown`: Cambia el propietario de un archivo.
- `tar`: Archiva archivos.
- `gzip`: Comprime o descomprime archivos.
- `unzip`: Extrae archivos de un archivo ZIP.
- `df`: Muestra el uso del espacio en disco.
- `du`: Muestra el uso del espacio de un directorio.
- `stat`: Muestra el estado de un archivo o sistema de archivos.
