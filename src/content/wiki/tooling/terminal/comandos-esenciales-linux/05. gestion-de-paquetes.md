# 05. Gestión de Paquetes

- `apt`: Herramienta avanzada de paquetes para gestionar paquetes de software.
- `apt-get`: Herramienta de línea de comandos para manejar paquetes.
- `yum`: Gestor de paquetes para distribuciones de Linux basadas en RPM.
- `dnf`: Gestor de paquetes de nueva generación para distribuciones de Linux basadas en RPM.
